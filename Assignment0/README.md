## Usage
| Keys  | Description                              |
| ----- | ---------------------------------------- |
| w/s   | Traslate y-axis                          |
| a/d   | Traslate x-axis                          |
| z/x   | Scale                                    |
| r/t   | Rotate                                   |
| Space | Toggle perspective/orthogonal projection |

Color values and other constants were gathered from [the requirements video](https://www.youtube.com/watch?v=5d6JBqpGnrE)
